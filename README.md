# SSM_To_Mall

## 简介
基于SSM（SpringMVC+Spring+MyBatis）通用框架，开发Java通用型电商后台系统

## 特点

图片开源工具 Thumbnailator、证件组件Kaptcha、Shell、 guava缓存、FTP服务、对接支付宝、二维码生成、 Jackson、joda、注解

## 功能

    通用商城后台
    缓存处理
    FTP服务
    二维码生成


## 配置

    datasource.properties 数据库配置
